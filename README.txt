== Description ==
Набор полезных сниппетов для быстрого запуска сайта на WordPress. Включает шорткоды, интеграции с CRM, аналитику, SEO-улучшения и многое другое.

**Основные возможности:**

= Шорткоды =
* [domain] - вывод домена сайта
* [url] - вывод URL сайта
* [page_content] - вставка контента страницы по ID или slug
* [current_year] - вывод текущего года
* [sf] - универсальный шорткод для вывода полей ACF

= Аналитика и отслеживание =
* Яндекс.Метрика - счетчик + отслеживание событий
* VK Pixel (Top.Mail.Ru) - интеграция с рекламным пикселем VK
* Sourcebuster.js - определение источника трафика + подмена телефонов
* UTM Tracker - сохранение UTM-меток в сессию и cookies

= Интеграции =
* Contact Form 7 → Битрикс24 CRM
* Contact Form 7 → Telegram
* SMTP настройки для отправки почты

= SEO =
* Подмена H1 из ACF полей для записей и таксономий
* Очистка заголовков архивов от префиксов

= Расширения для темы Blocksy =
* Дополнительные социальные сети (Дзен, Rutube)
* Кастомные хлебные крошки с микроразметкой Schema.org

= Улучшения админки =
* Фильтры по таксономиям для CPT
* Колонка с миниатюрами во всех списках записей
* IMask - маска ввода телефона

= REST API =
* Эндпоинт для ACF Options
* Поддержка Permalink Manager Pro
* Поля Yoast SEO в REST API

== Installation ==
1. Загрузите папку `rb-snippets` в каталог `/wp-content/plugins/`.
2. Активируйте плагин через меню 'Плагины' в WordPress.
3. Перейдите в Настройки → RB Snippets для просмотра справки.
4. Настройте интеграции в ACF Options → Настройки сайта.
