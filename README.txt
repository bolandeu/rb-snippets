== Description ==
Плагин позволяет выводить контент любой страницы или записи, а также отдельные блоки Gutenberg, используя шорткод [page_content].

== Installation ==
1. Загрузите папку `rb-snippets` в каталог `/wp-content/plugins/`.
2. Активируйте плагин через меню 'Плагины' в WordPress.
